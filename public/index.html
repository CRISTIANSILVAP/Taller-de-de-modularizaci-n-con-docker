Bienvenido al MicroFramework

Este es un servidor HTTP ligero implementado en Java sin Spring, con soporte para archivos estáticos (HTML, PNG, etc) y endpoints dinámicos.

Endpoints disponibles:

Home (índice) Saludo personalizado Página HTML adicional

Características

✓ Anotaciones @RestController y @GetMapping

✓ Procesamiento de parámetros @RequestParam

✓ Servicio de archivos estáticos (HTML, PNG, CSS, etc)

✓ Concurrencia con pool de hilos

✓ Apagado elegante

curl http://localhost:8080/
curl http://localhost:8080/greeting?name=Usuario
curl http://localhost:8080/page.html
