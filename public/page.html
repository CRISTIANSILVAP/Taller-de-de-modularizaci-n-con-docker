Página Estática HTML

Este archivo HTML es servido directamente por el servidor desde la carpeta public/.

Soporte de Archivos Estáticos

El servidor ahora puede entregar:

  • Archivos HTML (.html)
  • Imágenes PNG (.png)
  • Imágenes JPEG (.jpg, .jpeg)
  • Hojas de estilo CSS (.css)
  • Otros archivos estáticos

Cómo usar

  1. Coloca archivos en la carpeta public/
  2. Accede a ellos en tu navegador: http://localhost:8080/archivo.html
  3. El servidor detecta automáticamente el tipo MIME
